## Part A · 10 条公理的展开

### 公理 1：时间不是白来的（002、005）

**论证**：讲者在 002 的核心论点：人浪费时间最大的根本原因是「眼睛一睁又白捡一天」的潜意识。一旦把每段时间换算成「应得收入」（按真实时薪或理想时薪），时间就从「空气」变成「负债」，自动产生回本压力。

**算法**：
```
真实时薪 = 去年总收入 / 去年实际工作分钟数
理想时薪 = 你给自己定的目标价（讲者举例：纳瓦尔自定 5000 美元/小时；讲者自己取 800 元/小时 ≈ 13.33 元/分钟，激进口径取至每分钟 583 元）
日时间负债（已利用） = 当日实际工作分钟 × 单位价
日时间负债（总） = 24 × 60 × 单位价  // 把睡觉刷手机也算进去
日时间净值 = 当日产出价值 − 日时间负债
```

**反例 / 边界**：
- 不是说每分钟都要变现。是说每分钟的**机会成本**要被你看见。
- 休息 / 陪家人不是错；错的是不知道这段时间值多少。
- 真实时薪太低不必焦虑，定理想时薪用来筛事就够了。

### 公理 2：负债不是坏事，主动负债才是工具（002）

**论证**：被动负债（房贷、消费贷）让你被生活推着跑；主动负债（你给自己的时间债、项目债）则反过来约束你——一旦写进账，「回本需求」就会自动筛掉低价值任务、防止半途放弃。讲者：「一旦这个东西能够入账，你就必须严肃对待它。」

**反例**：盲目加杠杆买跌价资产 ≠ 主动负债，那是被动负债的伪装。判断标准：这笔债是逼你产出，还是逼你消费？前者主动，后者被动。

### 公理 3：买得起 ≠ 配得上（003、006，体系核心）

**两套定义并列对比**：

| 维度 | 买得起 | 配得上 |
|---|---|---|
| 数据来源 | 当下账户余额 / 当月收入 | **稳定被动收入** + 稳定理财利息 |
| 时间视角 | 此刻 | 长期、可持续 |
| 风险考虑 | 无 | 必须扛住生病 / 失业 / 收入归零 |
| 自由度 | 可以拿命换 | 不打扰你、不需要你亲自参与 |

**配得上的严格定义**（讲者 003 原话提炼）：
> 你到底配得上什么，严格的定义就是那个源源不断的、持续不断带来的稳定的收入能带给你多少，那就是你配得上花的钱。

**反例 / 常见混淆**：
- 「我月入 20 万，难道还不配花一万的包？」→ 错。月入 20 万只代表买得起；如果这 20 万明年没了，包是配不上的。
- 「我有 100 万存款 ≠ 配得上 100 万的生活」。讲者算法：100 万 × 4% / 12 ≈ 月 3333 元——这才是配得上。

### 公理 4：用配得上规划生活，不用买得起规划（003、006）

**论证**：从财务规划第一步就以「赚多少花多少」做基准，是绝大多数中产终身困局的起点。正确做法：
1. 设一个「不幸福但不痛苦」的目标生活（例：月 4000）。
2. 工资 1 万 → 4000 进生活账户，**6000 直接进资产积累账户，打死不动**。
3. 资产产生的被动收入超过 4000 → 才允许讨论升级生活标准。

**反例**：「我先享受当下，攒钱以后再说」→ 讲者明示这是中产陷阱核心机制。生活成本一旦提上去，**降回来比戒毒还难**（涉及家人、孩子、社会面子）。

---

### 公理 5：最优生活曲线（003）

**形状**：快速升到舒适点（短斜上）→ 长期纯平、零波动（长水平）→ 被动收入超线后才允许极缓上升（缓斜上）。

**反例**：常见错误曲线是阶梯持续上升（每涨一次工资就升一级生活）。讲者反复强调：**不要按月度循环**。月赚月花永远没有积累。

**边界**：「纯平」不是节衣缩食，是**主动钉死成本**。可以吃好住好——只要这个「好」是当年定下来的那个标准，不变就行。

### 公理 6：凡持有皆出血（002、006）

**完整算法**：
```
持有出血 = 单价 × (1 − 残值率) / 持有天数
续费出血 = 月续费金额 / 30
体验出血 = 单次金额 × 年频次 / 365
日均总出血 = Σ(以上三类)
```

**讲者举的具体数字**：
- iPhone 14765 元，2 年使用，残值 40% → 14765 × 0.6 / 730 ≈ 12.14 元/日
- 200 万自住房，10 年使用，残值假设 0 → 200 万 / 3650 ≈ 548 元/日（讲者讲座中也给过 5500/天 的更激进版本——口径不同）
- 50 万车开 10 年残值 5 万 → 45 万 / 3650 ≈ 123 元/日

**反例 / 边界**：
- 黄金、能持续升值的资产：残值 ≥ 100%，出血为 0 或负，可单列「资产收益」。
- 自住房残值率假设是关键变量；讲者倾向**保守取 0**（10 年后流动性归零的判断）。

### 公理 7：用行动定义意愿（002、贯穿）

**论证**：「想赚钱」=不想赚钱；「正在赚钱」=想赚钱。任何「我打算」「我准备」「我之后会」都视作零。**只有今天落到表格里的那一行才算。**

**应用**：当用户说「我想做财务系统」，唯一回应是：**「打开表格，现在就开始填 Phase 0 的六项。填完回来。」** 不要讨论方法、不要解释为什么、不要鼓励——只要动作。

---

### 公理 8：娱乐化是对严肃问题的解构（004）

**论证**：梗的传播效率比严肃论述高 1000 倍，但代价是把严肃问题（贫富、阶级、学历）变成段子，让所有人停止思考。讲者原话：**「梗文化压根就不能够碰。」**

**反例 / 边界**：
- 不依托严肃议题的梗（纯娱乐、纯生活）→ 影响小，不强禁。
- 但凡涉及人群对立、贬低、阶级 → 红灯。

---

### 公理 9：高级人设的反噬（004）

**论证**：人设杠杆越高，破裂时反噬越彻底。雷军的「真诚 / 真情」人设是最高杠杆——一旦被认定不真诚，所有过往真话都被解读为更高级演技，**且无解**。

**普通人对应启示**：
- 不要立「我永远正确」「我从没亏过」「我什么都懂」型人设。
- 主动暴露弱点 / 失误，**降低杠杆**，破裂时缓冲带就大。
- 户子案例的并行启示：通过对立框架建立的内容人设，最终会被自己的框架打入对立面。

---

### 公理 10：30 人以上的群没有高质量讨论（002）

**论证**：人数超过 30，信息密度必降；高质量讨论几乎不可能发生。继续守群有两种解释：群里有强者你想截获他偶发的发言（罕见、合理），或你筛选能力太弱（常见、应退）。

**应用**：用户抱怨「群里乱」「群里都是伸手党」→ 不要诊断群，诊断他为什么还在守这个群。

## Part B · 财务系统落地步骤（005、006 主线）

### Step 1 · 时间记录（005）

参考刘比歇夫时间记录法，每条记录五层信息：

| 字段 | 内容 |
|---|---|
| 起止时间 | 开始 HH:MM、结束 HH:MM |
| 任务内容 | 一句话写完做了什么 |
| 一级分类 | 工作 / 生活 / 娱乐 / 学习 / 休息 |
| 二级项目 | 具体属于哪个项目（用于后续 ROI 反摊） |
| 工作量 / AI 占比 | 完成单位 + AI 富能比例（0–100%） |
| 自评价值 | 1–10 分；当日产出价值估算（元） |
| 完成 1 分钟内复盘 | 备注 + 优化建议 |

**工具提示**：讲者用快捷键（Shift+Ctrl+T）调出记录表。任何方案都行，关键是**摩擦力 ≤5 秒**——超过 5 秒一定会放弃。

### Step 2 · 时间负债 / 净值算式（005）

```
单位时间价 = 去年总收入 / 去年总工作分钟  # 真实
or         = 你定的理想价                  # 理想
日时间负债（已利用） = Σ(当日工作分钟) × 单位时间价
日时间负债（总）     = 1440 × 单位时间价
日时间净值           = 当日产出价值 − 日时间负债
```

每日折线图盯三条线：**已利用负债 / 总负债 / 净值**。净值长期为负 = 再忙也是亏。

### Step 4 · 收入端三件套（006）

#### 4.1 主动收入

工资 / 接项目 / 直播 / 客单……按月入账，**不进配得上的分母**，只做参考。

#### 4.2 稳定理财日志

每日记录：余额宝、银行活期、微信、稳定币（USDT 利息）等。USDT 按当日汇率换成人民币。**这部分是「另一个永生不灭的你」的核心。**

#### 4.3 系统性 / 投资性收入

股票分红、企业股份分红、定投收益、版税、广告分成……每项打两个系数：

| 项目 | 年化金额 | 自由度系数 | 稳定性系数 | 加权年金 |
|---|---|---|---|---|
| 定投股票收益 | 30000 | 90% | 70% | 18900 |
| 全职炒股收益 | 80000 | 20% | 40% | 6400 |
| 比特币收益 | 100000 | 90% | 30% | 27000 |
| 书籍版税 | 5000 | 100% | 95% | 4750 |

**加权年金** = 金额 × 自由度 × 稳定性。这才是配得上分母里能用的钱。

### Step 5 · 配得上判定（每日跑一次）

```
日均稳定被动收入 = (Σ稳定理财年化 + Σ加权年金) / 365
日均总出血       = 持有 + 续费 + 体验
日均配得上生活成本（人均） = (日均稳定被动收入 − 日均总出血) / 家庭人数
```

判定：
- 该数 **≥** 你设定的目标日均生活标准 → **绿灯**：你目前真的配得上。
- 该数 **<** 目标 → **红灯**：你正在用「另一个自己」的命换今天的舒服。

### Step 6 · 五阶段定位（006）

| 阶段 | 标志 | 任务 |
|---|---|---|
| 1 完全负 | 工资 < 出血 | 砍出血，开源 |
| 2 工资覆盖出血 | 工资 ≥ 出血，被动收入 ≈ 0 | 钉死生活成本，把增量塞资产 |
| 3 主动建被动收入 | 资产端开始有利息 | 持续积累，不动用 |
| 4 初级 FIRE | 被动收入 ≥ 出血 | 守住，开始考虑生活缓升 |
| 5 高级 FIRE | 被动收入 ≥ 出血 + 时间债务 | 自由 |

**关键提醒**：90% 来咨询的人**自我感觉在 4，实际在 2**。区别只看一个数：「工资明天归零，你能维持当前生活几个月？」

## Part C · 配得上判断框架（003、006 综合）

针对**任何一笔具体消费 / 一种生活方式 / 一项资产决策**，按以下流程跑：

### 输入

1. 这件事的**全口径年化成本**：一次性出血 + 续费 + 它带来的体验出血。
   - 例：买一辆车 → 折旧出血 + 保养保险 + 加油 + 出去玩多消耗的体验。
2. 你的**配得上门槛**：当前稳定被动收入年化 × 自由度 × 稳定性。

### 判定

```
if 全口径年化成本 < 配得上门槛:
    return 绿灯
else:
    return 红灯
```

### 三道追问（任何一问答不上就强制红灯）

1. **稳定性**：这笔覆盖钱明年还在吗？后年呢？
2. **自由度**：是不打扰你的钱，还是要拿命换的钱？
3. **风险**：这笔覆盖钱归零的那个月，这项支出能停吗？

### 处方

- 绿灯 → 买。但写进出血表。
- 红灯 → 不买。给替代方案：
  - 降配到配得上的版本（清迈别墅 vs 杭州别墅，讲者亲身案例）。
  - 用「地理套利」换体验（同样体验在另一座城市价格 1/5）。
  - 暂停 N 个月，等被动收入升上来再说。

### 配得上的反面：超前消费的新定义

> 凡是花了你配不上的东西，都叫超前消费——不论你兜里有多少钱。

这是讲者 003 全集最重要的一句改写。把它贴在用户的钱包上。
